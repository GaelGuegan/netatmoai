# Authentication
In order to have access to netatmo API, credentials are need for this app.
It has to be written in the file netatmo_credentials, with the following format:
```json
{
    "CLIENT_ID": "xxxxx",
    "CLIENT_SECRET": "xxxxx",
    "REFRESH_TOKEN":  "xxxx|xxxx"
}
```
